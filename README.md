## 🚀 Using the interface

When you open r2filebox, you will see a clean screen. The app focuses on two main tasks: uploading files and managing your existing uploads.

### How to upload a file
1. Click the "Add File" button located at the center of the window.
2. Select the file you want to share from your computer.
3. Choose an expiration time for the file. This ensures the system cleans up the data automatically.
4. Add an extraction code if you want to keep the link private.
5. Click the "Upload" button.

The app shows a progress bar while it sends your data. Once the process ends, the app gives you a unique link. You can copy this link and send it to anyone who needs the file. They can open the link in any web browser to view or download the content.

### Using extraction codes
For extra safety, add a code during the upload process. The person receiving the file must enter this code to download the content. This prevents random people from accessing your shared items if they happen to find your link.

## ☁️ Understanding the cloud features

r2filebox works differently than standard file sharing services. It uses serverless technology. This means the app runs on a distributed cloud network rather than a single computer. 

*   **Resumable Uploads:** If your internet connection drops while you send a large file, the app remembers where it stopped. It finishes the transfer once your connection returns.
*   **Temporary Storage:** The system removes files once they pass their expiration date. This keeps your storage space clean.
*   **Anonymity:** The app does not require a user account. You share files without linking your identity to your activity.

## 🛠 Troubleshooting common issues

If you run into trouble, follow these simple steps to fix the issue.

*   **App won't open:** Make sure you have the latest version of Windows installed. Restart your computer and try again.
*   **Upload fails:** Check your internet connection. Large files can take time to upload depending on your internet speed. Wait for the progress bar to reach the end before closing the app.
*   **Link doesn't work:** Check if the file has expired. If the link was deleted, you must upload the file again.
*   **Slow performance:** If you share many files at once, the system might need a few moments to organize the data. Give it extra time to complete the indexing process.
